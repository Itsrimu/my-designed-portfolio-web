# Portfolio Website

My  personal portfolio website built with React.

## Features

- Responsive design
- Hero section with animated text
- Downloadable resume
- Contact and social links

## Getting Started

1. **Install dependencies:**
   ```sh
   npm install
   ```

2. **Run the development server:**
   ```sh
   npm start
   ```

3. **Build for production:**
   ```sh
   npm run build
   ```
